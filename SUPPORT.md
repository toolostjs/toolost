# Support

## Getting help

- For usage questions and ideas, use GitHub Discussions:
  https://github.com/toolostjs/toolost/discussions
- For bug reports and feature requests, use the issue templates:
  https://github.com/toolostjs/toolost/issues/new/choose

## Security issues

For sensitive security reports, follow [SECURITY.md](./SECURITY.md) and submit
through GitHub Security Advisories.

## Maintainer expectations

This is a community-maintained project. Response times may vary depending on
maintainer availability.
