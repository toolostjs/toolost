# toolost

Unofficial JavaScript/TypeScript SDK for the Too Lost Developer API.

⚠️ This package is currently under active development and not ready for production use.

## Status

🚧 Work in progress — initial release coming soon.

## Installation

```bash
npm install toolost
# or
yarn add toolost
# or
pnpm add toolost
# or
bun add toolost
```
